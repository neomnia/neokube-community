# Agents

This directory contains example agent definitions and reusable templates for NeoKube.

## What is a NeoKube Agent?

A NeoKube agent is an autonomous unit that:
- Receives a trigger (webhook, schedule, user input, API call)
- Executes a sequence of actions (LLM calls, API requests, data transforms)
- Returns a structured output or triggers a downstream workflow
