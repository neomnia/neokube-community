# NeoKube Community

> **Open experimentation layer for NeoKube — the agentic AI orchestration platform by [Neomnia Studio](https://neomnia.net).**

---

## What is NeoKube?

NeoKube is an agentic AI platform designed to automate business workflows, integrate deeply with client stacks (CRM, ERP, document management), and orchestrate AI agents in production environments.

This repository is the **community edition** — open for developers, researchers, and integrators who want to:
- Explore NeoKube's architecture and agent patterns
- Build local prototypes and proof-of-concepts
- Contribute to the open layer of the platform

---

## Repository Structure

```
neokube-community/
├── agents/          # Example agent definitions and templates
├── workflows/       # Sample workflow patterns (CRM, admin, support)
├── connectors/      # Starter connectors (REST, webhooks, MCP)
├── docs/            # Architecture documentation
└── examples/        # End-to-end usage examples
```

> ⚠️ This repository is under active development. Starter content is being added progressively.
