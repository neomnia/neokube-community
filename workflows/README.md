# Workflows

This directory contains sample workflow patterns built on NeoKube.

## Workflow Patterns

| Pattern | Description | Status |
|---|---|---|
| CRM follow-up | Automated relaunches and lead nurturing | Coming soon |
| Document intake | Collect, validate, and route administrative documents | Coming soon |
| Onboarding flow | Client onboarding with step tracking and reminders | Coming soon |
| Support triage | Classify and route incoming support requests | Coming soon |

Workflows are defined as YAML or Python orchestration files compatible with NeoKube's runtime.
