## Key Principles

- **Agent-first** вҖ” every business action is encapsulated in a traceable, auditable agent
- **Stack-agnostic** вҖ” connects to any CRM, ERP, or document system via connectors
- **Sovereign by design** вҖ” runs on European infrastructure, compatible with GDPR requirements
- **Subscription-based maintenance** вҖ” deployed once, maintained continuously
